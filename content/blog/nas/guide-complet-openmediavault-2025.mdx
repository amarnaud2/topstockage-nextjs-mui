# OpenMediaVault : Guide complet pour monter votre NAS en 2025

OpenMediaVault (OMV) est une solution NAS open source basée sur Debian, idéale pour créer un serveur de stockage domestique ou petit bureau. Découvrez comment monter votre propre NAS avec OMV.

## Pourquoi choisir OpenMediaVault ?

### Points forts
- Basé sur Debian (stabilité)
- Interface web intuitive
- Légèreté du système
- Large choix de plugins
- Communauté active
- Parfait pour débutants

### Prérequis techniques
- Connaissances Linux basiques
- Notions de réseau
- Compréhension du RAID

## Services et applications populaires

### Multimédia
1. **Plex**
```yaml
version: '3'
services:
  plex:
    image: linuxserver/plex
    network_mode: host
    environment:
      - PUID=1000
      - PGID=1000
    volumes:
      - /chemin/config:/config
      - /chemin/media:/media
```

### Cloud personnel
1. **Nextcloud**
```yaml
version: '3'
services:
  nextcloud:
```
